```
    例: ホースに亀裂あり、扉が開きにくい
  
  💬 備考
    気づいたことを自由に記入
```

### 10. よくある質問（Q&A）（3スライド）

#### 10-1. アプリの使い方に関する質問
```
Q1: ログインは必要ですか?
A1: いいえ、ログイン不要です。
   記録時に名前を選択するだけでOKです。

Q2: インターネットに繋がっていないと使えませんか?
A2: はい、インターネット接続が必要です。
   Wi-Fiまたはモバイルデータ通信をONにしてください。

Q3: パソコンからも使えますか?
A3: はい、使えます！
   同じURL（https://oifire1.pages.dev）を
   パソコンのブラウザで開いてください。

Q4: アプリをダウンロードする必要はありますか?
A4: いいえ、ダウンロード不要です。
   ブラウザで開くだけで使えます。

Q5: データは誰が見られますか?
A5: URLを知っている人は誰でも見られます。
   団内での共有を前提としています。
```

#### 10-2. 操作に関する質問
```
Q6: 記録を間違えて保存してしまいました
A6: 各記録には編集・削除ボタンがあります。
   該当の記録を開いて修正または削除してください。

Q7: 前回の記録をコピーして使いたい
A7: 活動記録の場合、前回のメーター情報は
   自動で入力されます。
   その他の項目は手動で入力してください。

Q8: 写真は保存できますか?
A8: 現在のバージョンでは写真保存機能は
   ありません。将来のアップデートで
   追加予定です。

Q9: 複数の人が同時に記録できますか?
A9: はい、同時に使えます。
   データはリアルタイムで共有されます。

Q10: 削除したデータは復元できますか?
A10: いいえ、削除したデータは復元できません。
    削除前に必ず確認してください。
```

#### 10-3. 紙で見たい方への対応
```
Q11: このマニュアルを紙で見たいのですが?
A11: はい、印刷できます！

印刷方法（パソコンから）:
ステップ1: このスライドをダウンロード
ステップ2: PowerPointまたはPDFで開く
ステップ3: 「ファイル」→「印刷」を選択
ステップ4: 「配布資料」を選択
  → 1ページに2～4スライド表示がおすすめ
ステップ5: 印刷実行

印刷方法（スマホから）:
ステップ1: PDFをスマホにダウンロード
ステップ2: コンビニのプリントサービスを利用
  例: セブンイレブン「ネットプリント」
     ローソン・ファミマ「ネットワークプリント」
ステップ3: アプリで印刷予約
ステップ4: コンビニで印刷

💡 ヒント:
印刷したマニュアルは、消防団の詰所に
置いておくと便利です！
```

### 11. 困ったときの対処法（2スライド）

#### 11-1. トラブルシューティング
```

✅ 対処法:
• インターネット接続を確認
• ブラウザを閉じて再度開く
• URLが正しいか確認
  https://oifire1.pages.dev

❌ ボタンを押しても反応しない
✅ 対処法:
• 軽くタップしてみる（強く押さない）
• ページを再読み込み
  → 画面を引っ張って更新
• ブラウザのキャッシュをクリア

❌ 「エラーが発生しました」と表示される
✅ 対処法:
• 必須項目（*マーク）を入力したか確認
• 選択肢から選ぶ項目は必ず選択
• 数字を入力する項目に文字を入れていないか確認
• エラーメッセージをよく読む
  → 何が問題か表示されます
```

#### 11-2. その他の問題
```

• ブラウザを閉じて再度開く

❌ CSVファイルがうまくアップロードできない

このアプリはブラウザ上で動作するため
スマホの容量は使いません。
ただし、CSVファイルをダウンロードする場合は
少し容量が必要です。

それでも解決しない場合:
```
